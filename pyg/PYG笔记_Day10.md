**视频信息**
```
视频名称: 16.搜索条件构建-撤销查询条件
视频时长: 04:26
```
**小节内容**
```
delete 是js 提供的语法 
delete 操作符用于删除对象的某个属性；如果没有指向这个属性的引用，那它最终会被释放。
```

```js
补充:
	js 中如果一个变量被 var 声明了则不能删除,如果没有加var 则可以删除
var i =1;
delete i; // 不能删除
alert(i);// 结果为 1
j =1;
delete j; // 可以删除
alert(j);// 结果后台报错: 变量未定义

```

js补充: == 和 === 的区别

```

alert(1=="1");// true    == 表示 值相等即可
alert(1==="1"); // false === 要求值和类型都相同才是true
alert(1===1); //true
```
